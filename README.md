# SectionsTableView.Swift

Simple solution for static or paginating table views.

## Installation

### Swift Package Manager

```swift
dependencies: [
    .package(url: "https://github.com/horizontalsystems/SectionsTableView.Swift.git", .upToNextMajor(from: "1.0.0"))
]
```
